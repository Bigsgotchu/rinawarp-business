## Escalation Responses

### If someone persists on "overpriced"

Look at their profile. If they're a student or indie dev:
"Completely understand. Free tier exists exactly for this. Upgrade only if/when it saves you time worth $29/month."

If they're clearly a enterprise/VC type:
"Pricing reflects real development costs + hosting. If you need enterprise features, we can discuss custom pricing."

### If someone questions lifetime legitimacy

"Fair skepticism. Lifetime means what it says: local product + features at time of purchase. No hidden fees, no forced subscriptions later. We depend on word-of-mouth, not lock-in."

### If someone wants features for free

"Free tier is intentionally useful, not crippled. If you need more, the agent is doing real work (API calls, compute, maintenance) that costs money to provide."
