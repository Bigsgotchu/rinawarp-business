# Security Policy

## Supported Versions

We support the latest `main` and the most recent tagged release.

## Reporting a Vulnerability

- Email: **security@rinawarp.com** (preferred)
- Alternatively: open a **Private** security advisory on GitHub (Repository → Security → Advisories)

Please include:

- Affected version/commit, environment, and reproduction steps
- Impact assessment (data exposure, RCE, DoS, etc.)
- Proof-of-concept (minimal)

We aim to acknowledge within **48 hours** and provide a timeline within **5 business days**.

## Disclosure

We practice **coordinated disclosure**. We'll work with you on timelines and public communication. Credit will be granted unless anonymity is requested.

## Out of Scope

- Social engineering
- Physical attacks
- Issues requiring rooted/jailbroken or non-default platforms
