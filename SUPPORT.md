# Support

## Self-Serve

- Troubleshooting: see **RELEASING.md** and CI logs
- Common tasks:
  - `pnpm ship` – full pipeline
  - `pnpm format:fix && pnpm lint:fix` – quick cleanup
  - `pnpm smoke:preload` / `:packaged` – Electron preload verification

## Contact

- Bugs: GitHub Issues
- Security: security@rinawarp.com (see SECURITY.md)
- Priority incidents (production outage): **on-call@rinawarp.com**

## Response Targets

- Normal issues: 3 business days
- Production-impacting issues: within same business day
