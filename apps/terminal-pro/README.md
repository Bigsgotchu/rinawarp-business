# RinaWarp Terminal Pro VS Code Extension

üöÄ **AI-powered development assistant by RinaWarp**

This extension integrates RinaWarp Terminal Pro directly into VS Code, providing a seamless development experience with embedded tools, one-click deployment, and AI-powered suggestions.

# ‚ú® Features

# üéØ **Core Features**

- **Terminal Pro Integration**: Launch RinaWarp Terminal Pro directly from VS Code

- **Dev Dashboard**: Embedded web-based development dashboard

- **One-Click Deploy**: Deploy your entire stack with a single command

- **AI Fix Suggestions**: Get intelligent debugging and optimization suggestions

- **Plugin Sync**: Automatically sync and manage your development plugins

# üé® **UI Integration**

- **Sidebar Panel**: Dedicated RinaWarp panel in the explorer view

- **Status Bar**: Quick access status bar indicator

- **Command Palette**: Easy access via Ctrl+Shift+R and other keybindings

- **Webview Support**: Embedded HTML interfaces for rich interactions

# ‚ö° **Productivity Features**

- **Auto-scan**: Automatically detect and suggest fixes for common issues

- **Project Intelligence**: Context-aware suggestions based on your project structure

# üöÄ Quick Start

# Prerequisites

- VS Code 1.70.0 or later

- Node.js 14+ installed

- RinaWarp project in your workspace

# Installation

1. **Install Extension**:

   ```bash
    cd vscode-extension
    npm install
    npm run compile
    code --install-extension dist/rinawarp-terminal-pro-1.0.0.vsix
```css

1. **Configure Extension**:
    - Press `Ctrl+,` to open Settings
    - Search for "RinaWarp"
    - Configure paths and URLs as needed

1. **First Launch**:
    - Press `Ctrl+Shift+R` to open Dev Dashboard
    - Or use the sidebar RinaWarp panel
