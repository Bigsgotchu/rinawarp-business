### Pro Tip 1: Quick Session Switching

```bash
# Save your current session
:session save development

:session load development
```

### Pro Tip 2: AI-Powered Command History

```bash
# Ask AI to analyze your command history
:ai history

```

```bash

Ctrl+Shift+E

Ctrl+Shift+Right

# Run tail command in the split
$ tail -f /var/log/syslog
```
