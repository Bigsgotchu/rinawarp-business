# RinaWarp Brain

RinaWarp Brain is a VS Code extension designed to streamline cryptographic plan management, verification, and execution workflows for developers.

## Features

- **Cryptographic Plan Hashing** – Securely verify and manage plans.
- **Approval Flow** – Plan → Preview → Approve → Execute workflow.
- **Session Timeline** – Track all sessions and exports in a timeline view.
- **Developer Tools Integration** – Commands and automation directly in VS Code.

## Installation

1. Open VS Code.
2. Go to Extensions (`Ctrl+Shift+X` / `Cmd+Shift+X`).
3. Search for **RinaWarp Brain** and install.
4. Reload VS Code if prompted.

## Usage

- Access commands via the **Command Palette (`Ctrl+Shift+P`)**.
- Configure settings in `Settings > Extensions > RinaWarp Brain`.

## Commands

| Command | Description |
|---------|-------------|
| `RinaWarp: Start Plan` | Initialize a new plan workflow |
| `RinaWarp: Preview Plan` | Preview current plan details |
| `RinaWarp: Approve Plan` | Approve a plan for execution |
| `RinaWarp: Execute Plan` | Execute approved plan workflow |
| `RinaWarp: Verify Plan` | Verify cryptographic integrity of plan |
