# [2.0.0] - 2025-11-29

# Added
 - RinaWarp Enterprise v2 documentation suite (18+ structured markdown files).
 - Full RinaWarp Docs site powered by MkDocs Material.
 - Docs CI/CD pipeline with GitHub Actions + Netlify deployment.
 - Printable "RinaWarp Enterprise Manual v2.0" PDF build script.
 - Formal security stack documentation: UFW, Fail2ban, CrowdSec, NGINX, SSL.
 - Release management and versioning guidelines.
 - "Docs" navigation link added to main website.
 - Cloudflare DNS configuration guide for docs subdomain.
# Changed
 - Standardized documentation naming (01-Architecture-Overview, etc.).
 - Updated deployment flows to use automated CI/CD pipeline.
 - Cleaned up markdown across the repo using automated fixes.
 - Website navigation updated to include Docs link pointing to docs.rinawarptech.com.
# Fixed

 - Multiple minor markdown issues (bare URLs, inline HTML, trailing blank lines).
 - Website navigation consistency across all pages.
