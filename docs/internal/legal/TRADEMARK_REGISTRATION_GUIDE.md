### Step 1: Trademark Search

**Before filing, conduct comprehensive searches:**

1. **USPTO Trademark Search:**
   - Go to: <https://tmsearch.uspto.gov/>
   - Search for: "RinaWarp", "RinaWarp Terminal", "RinaWarp Pro"
   - Check for conflicts or similar marks

2. **Google Search:**
   - Search: "RinaWarp trademark"
   - Search: "RinaWarp Terminal Pro"
   - Check for existing usage

3. **Domain Search:**
   - Check: rinawarp.com, rinawarptech.com
   - Check: rinawarp-terminal.com
   - Check: rinawarp-pro.com

### Step 2: Trademark Application (USPTO)

**Online Filing:**

1. Go to: <https://www.uspto.gov/trademarks-application-process>
2. Use TEAS (Trademark Electronic Application System)
3. File under: **Intent to Use** (since products are in development)

**Required Information:**

- **Owner**: RinaWarp Technologies, LLC
- **Address**: [Your business address]
- **Goods/Services**:
  - Computer software for terminal emulation
  - Computer software for AI-powered video generation
  - Computer software for music video creation
  - Software as a Service (SaaS)

**Filing Fees:**

- TEAS Standard: $350 per class
- TEAS Plus: $250 per class (if you use pre-approved descriptions)

### Step 3: International Registration

**Madrid Protocol:**

- File through USPTO for international protection
- Covers multiple countries with single application
- Additional fees per country

**Priority Countries:**

1. **United States** (Primary)
2. **European Union** (EUIPO)
3. **Canada** (CIPO)
4. **United Kingdom** (UKIPO)
5. **Australia** (IP Australia)
6. **Japan** (JPO)
