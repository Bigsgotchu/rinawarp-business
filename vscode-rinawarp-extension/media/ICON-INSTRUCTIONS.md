# RinaWarp Extension Icon Instructions

# Required Icons

You need to add these PNG icon files to the `media/` folder:

# 1. Main Icon (Required)

- **File**: `media/rinawarp-icon.png`

- **Size**: 32x32 pixels (recommended)

- **Purpose**: Extension icon in activity bar, file explorer, and extensions view

- **Description**: Your RinaWarp Terminal Pro logo

# 2. Dark Theme Icon (Optional)

- **File**: `media/rinawarp-icon-dark.png`

- **Size**: 32x32 pixels (recommended)

- **Purpose**: Alternative icon for dark VS Code themes

- **Description**: Dark variant of your logo

# Logo Requirements

- **Format**: PNG with transparency support

- **Recommended Size**: 32x32 pixels

- **Maximum Size**: 128x128 pixels

- **Background**: Transparent or VS Code will handle transparency

# Where to Get Your Logo

1. **Use Existing Logo**: If you have a RinaWarp logo, resize it to 32x32 pixels
2. **Create New Logo**: Design a simple, recognizable icon representing RinaWarp Terminal Pro
3. **Convert to PNG**: Ensure the final file is in PNG format

# Quick Start

For immediate testing, you can:

1. Use any 32x32 PNG image temporarily
2. Rename it to `rinawarp-icon.png`
3. Place it in the `media/` folder

# Logo Design Tips

- Keep it simple and recognizable at small sizes

- Use high contrast for visibility

- Avoid complex details that won't show at 32x32 pixels

- Consider using your brand colors

---

**Note**: The extension will work without custom icons, but the default VS Code icon will be used instead.
