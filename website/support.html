Support & Status

Realistic expectations. Clear channels. This is where you check the status of RinaWarp Terminal Pro™ and get help if something breaks.

What We Support Right Now

  • Questions about RinaWarp Terminal Pro scripts & usage
  • Help understanding the deployment & health-check scripts
  • Clarifications on docs, configuration, and expected behavior
  • Bug reports or unexpected errors in the official scripts

What's Still "Best Effort"

  • Custom infrastructure changes far outside the docs
  • Very old or unsupported OS versions
  • Third-party services you wire in yourself

Error Reports & Logs

When reporting a bug, include:

  • OS & version (e.g., Ubuntu 20.04 / Kali Rolling)
  • Command you ran (e.g., ./rinawarp-prod.sh)
  • Relevant log snippet from /var/log/rinawarp/launch/

Current Status

RinaWarp Terminal Pro™ is under active development. Security, deployment, and documentation are being hardened step-by-step, with real logs and health-checks running on your Oracle VM.

  • NGINX + SSL configuration: handled by setup_nginx_ssl.sh
  • Security stack: UFW, Fail2ban, CrowdSec (installed and active)
  • Health & integration tests: via health-check.sh and integration-test.sh
